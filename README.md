# Tracking Status Normalizer

A Python component that normalizes heterogeneous carrier tracking events into a unified shipment status taxonomy.

## Status

Early development. Version 0.1.0.

## Requirements

- Python 3.11 or higher

## Installation

Clone the repository and install in editable mode:

```bash
git clone https://github.com/<your-account>/tracking-status-normalizer.git
cd tracking-status-normalizer
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -e .[dev]
```
